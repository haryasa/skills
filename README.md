# Agent Skills

A collection of agent skills for solo developers and small dev teams. Works with 50+ coding agents — Claude Code, Cursor, Copilot, Windsurf, Cline, and more.

## Installation

```bash
npx skills add haryasa/skills
```

## Skills

### `article-reader`

Read and analyze articles from URLs or pasted content. Produces a structured brief with TL;DR, key takeaways, stats, actionable advice, notable quotes, and an honest read recommendation (SKIP / SKIM / READ FULLY).

**Triggers on:** shared URLs, pasted article text, "summarize this", "is this worth reading?"
